Network Infrastructure

Introduction

  • Hardware and software resources of an entire network
  • Provides the communication path and services between users, processes, applications, services and external networks/the Internet.
  • Interconnected group of computer systems linked by the various parts of a telecommunications architecture
  • Infrastructures can be either open or closed

Networking Hardware

  • Routers
  • Switches
  • LAN cards
  • Wireless routers
  • Cables

Networking Software

  • Network operations and management
  • Operating systems
  • Firewall
  • Network security applications

Network Service

  • DSL
  • Wireless protocols
  • IP addressing
  • T-1 Line

I-way

Information Superhighway will allow us to share information, to connect, and to communicate as a global community.

I-way

  • a physical network, an infrastructure of modern high-speed links
  • links everyone at home or office to everything else
  • in practicality it is identical to internet provided -connections are broadband -they are continuously running
  • as internet develops into I-way, changes will take place in infrastructure not in internet
